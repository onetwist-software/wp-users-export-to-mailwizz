Allows you to export your users to a MailWizz list of subscribers

== Description ==

Adds an entry under the Tools menu in the Wordpress admin area that allows you to export users to your [MailWizz Email Marketing Application](http://www.mailwizz.com/ "MailWizz Email Marketing Application") based on the
[PHP-SDK](https://github.com/twisted1919/mailwizz-php-sdk/ "PHP-SDK").  
Using this plugin you will be able to export to the MailWizz list the selected categories of Wordpress users.


== Installation ==  

1. Upload the folder `wp-users-export-to-mailwizz` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Tools menu from the admin area to use it
